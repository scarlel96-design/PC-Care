# PCCare 전체 UI/UX 개편 프로토타입

이 폴더는 실제 PCCare 기능 코드와 연결되지 않은 독립 시안이다. `index.html`을 브라우저로 열면 전체 메뉴와 시스템 케어의 검사 전·진행·결과·완료 상태를 확인할 수 있다.

## 설계 목표

- 기능 목록이 아니라 사용자 목표를 기준으로 `홈 / PC 최적화 / 보안 / 기록 및 복구 / 설정` 5개 영역으로 재구성
- 홈에서는 현재 상태와 가장 중요한 행동 하나를 먼저 제시
- 점검은 `검사 설정 → 검사 중 → 결과 검토 → 정리 완료`의 명확한 단계로 분리
- 결과는 전체 문제 개수만 강조하지 않고 확보 공간, 성능 영향, 안전도와 복구 가능 여부를 함께 표시
- 안전한 권장 항목은 기본 선택하고 환경에 따라 영향이 달라지는 항목은 `검토 필요`로 분리
- 보안 금고의 GC, 마이그레이션, 무결성 검사 같은 전문 기능은 고급 영역으로 접기
- 모든 변경 기록과 되돌리기를 `기록 및 복구` 한 화면으로 통합

## 구현 대상과의 매핑

| 프로토타입 | 기존 화면 |
| --- | --- |
| 홈 | `MacDashboardPage` |
| PC 최적화 | `UnifiedCarePage`, `SystemCareCenterPage`, 드라이버·오디오 진단 |
| 보안 | `SecureVaultCenterPage`, `SecureDeleteCenterPage` |
| 기록 및 복구 | `ReportPage`, `EvidenceExplorerPage`, 복구 기록 |
| 설정 | `SettingsPage`, `FeatureManagementPage`, 고급 도구 진입점 |

## 주의

화면 안의 수치와 결과는 UI 상태를 보여주기 위한 예시 데이터다. 검사, 정리, 암호화, 삭제, 복구 작업은 실행하지 않는다.

## 디자인 언어

- macOS: 넓은 여백, 반투명 상단 레이어, 섬세한 깊이와 집중된 기본 동작
- One UI 9 계열: 큰 제목, 넉넉한 터치 영역, 핵심 정보가 상단과 중앙에 먼저 오는 구조
- Windows 11: Mica 느낌의 배경, 둥근 카드, 익숙한 토글·진행 상태·복구 흐름
- 화려한 그라데이션과 빛 효과는 홈 상태 카드와 주요 완료 상태에만 사용하고 결과 목록과 설정은 차분하게 유지
- 파랑은 기본 동작, 초록은 안전·완료, 주황은 검토, 빨강은 되돌릴 수 없는 작업에만 사용
